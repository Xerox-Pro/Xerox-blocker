数学I

数と式

実数、無理数、複素数など数の概念と、式の展開、因数分解、方程式、不等式について学びます。

(a+b)^2 = a^2 + 2ab + b^2 x^2 - y^2 = (x+y)(x-y)

二次関数

二次関数のグラフ（放物線）、頂点、軸、最大値・最小値の求め方、二次方程式との関係を理解します。

y = ax^2 + bx + c (一般形) y = a(x-p)^2 + q (頂点形) 頂点: (p, q)

例えば、y = x2 - 4x + 3 のグラフは次のようになります。

図形と計量

三角比（sin, cos, tan）、正弦定理、余弦定理を用いて図形の辺の長さや角度を求めます。

正弦定理: a/sinA = b/sinB = c/sinC = 2R 余弦定理: a^2 = b^2 + c^2 - 2bc cosA

データの分析

データの散らばり具合を表す指標（平均値、中央値、最頻値、分散、標準偏差）や、相関関係について学びます。

分散: s^2 = (1/n) * Σ(xi - x̄)^2 標準偏差: s = √s^2

数学A

場合の数と確率

順列、組み合わせ、重複順列、重複組み合わせといった場合の数の数え方と、それらを用いた確率の計算方法を学びます。

nPr = n! / (n-r)! (順列) nCr = n! / (r! * (n-r)!) (組み合わせ)

図形の性質

三角形の性質（外心、内心、重心、垂心）、円の性質（接線、弦、方べきの定理）などを学びます。

メネラウスの定理, チェバの定理 方べきの定理: PA・PB = PC・PD (交わる2つの弦)

整数の性質

約数と倍数、素数、最大公約数、最小公倍数、ユークリッドの互除法、合同式について学びます。

a ≡ b (mod m) (合同式)

数学II

様々な式

分数式、恒等式、等式・不等式の証明、高次方程式、複素数と方程式について学びます。

複素数: z = a + bi (i は虚数単位)

図形と方程式

点と直線、円の方程式、軌跡と領域について学びます。

円の方程式: (x-a)^2 + (y-b)^2 = r^2 点と直線の距離: |Ax0 + By0 + C| / √(A^2 + B^2)

指数関数・対数関数

指数法則、指数関数、対数関数とその性質、グラフについて学びます。

log_a M = p ⇔ M = a^p log_a (MN) = log_a M + log_a N

微分法と積分法

導関数、接線の方程式、関数の増減、極値、定積分、不定積分、面積の計算について学びます。

f'(x) = lim(h→0) {f(x+h) - f(x)} / h ∫ x^n dx = (1/(n+1)) * x^(n+1) + C

数学B

数列

等差数列、等比数列、漸化式、数学的帰納法、Σ（シグマ）記号による和の計算について学びます。

等差数列の一般項: an = a1 + (n-1)d 等比数列の一般項: an = a1 * r^(n-1)

ベクトル

ベクトルの成分表示、内積、外積、位置ベクトル、直線や平面の方程式への応用を学びます。

ベクトルa = (ax, ay), ベクトルb = (bx, by) 内積: a・b = |a||b|cosθ = ax*bx + ay*by

確率分布と統計的な推測

二項分布、正規分布などの確率分布、期待値、分散、標本調査、推定、仮説検定の基礎を学びます。

期待値 E(X) = Σ xi * P(X=xi)
